# Problem Statement

Convert an `Int` `d` from decimal to binary

## Constrains

- Don't use languages utility functions

# Example 1:

**Input:** d = 2
**Output:** b = 10

# Example 2:

**Input:** d = -10
**Output:** b = -1010

# Example 3:

**Input:** d = 0
**Output:** b = 0

# Example 4:

**Input:** d = Int.MAX_VALUE
**Output:** b = 1111111111111111111111111111111

# Example 5:

**Input:** d = Int.MIN_VALUE
**Output:** b = -10000000000000000000000000000000

# Example 6:

**Input:** d = 1
**Output:** b = 1
