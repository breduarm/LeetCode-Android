# Problem Statement

Reverse the order of a t: String without using utilities function of the language
- E.g. t = "Hola mundo", result = "odnum aloH"

# Example 1:

**Input:** t = "Hola mundo"
**Output:** return = "odnum aloH"

# Example 2:

**Input:** t = ""
**Output:** return = ""

# Example 3:

**Input:** t = "     hello     "
**Output:** return = "     olleh     "
