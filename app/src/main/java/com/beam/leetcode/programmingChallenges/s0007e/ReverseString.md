# Problem Statement

Reverse the order of a `s: String` without using utilities function of the language
- E.g. `s = "Hola mundo", result = "odnum aloH"`

# Example 1:

**Input:** t = "Hello world"
**Output:** return = "dlrow olleH"

# Example 2:

**Input:** t = ""
**Output:** return = ""

# Example 3:

**Input:** t = "     hello     "
**Output:** return = "     olleh     "
